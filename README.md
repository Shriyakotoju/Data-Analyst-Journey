# Data-Analyst-Journey
My hands on journey learning Excel, SQL, and Power BI for Data Analyst roles.
This repository contains my hands on practice work while learning
Excel, SQL, and Power BI for a Data Analyst role.

## Excel – Day 1
Topics covered:
- Excel basics and formulas
- Text functions (TRIM, PROPER, LEFT, MID, RIGHT)
- Logical functions (IF, AND, OR, NESTED IF, IFS)
- Aggregation functions (SUM, COUNT, AVERAGE, SUMIF, SUMIFS)
- Lookup functions (VLOOKUP, XLOOKUP)
- Sorting, filtering, and tables

Excel – Day 2
=============
Topics covered:
- Data cleaning and text standardization
- Text functions for cleaning (TRIM, PROPER, UPPER)
- Handling missing values
- Analysis questions (unique customers, total sales, average sales, city-wise sales)
- Helper tables for city-wise aggregation

Excel – Day 3

Topics covered:

- PivotTables for business analysis
- Sales analysis by city and category
- Time-based sales trend analysis
- Creating helper columns for date analysis (Excel Online)
- Chart creation (Column and Line charts)
- Using slicers for interactive filtering
- Building a clean, interactive Excel dashboard

Tools used:
- Microsoft Excel
- GitHub
